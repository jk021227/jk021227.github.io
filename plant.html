Team Members: Alex Hmitti, Jhon Kim, Aaron Stein, and Rafael Nadal-Scala

üìñ Description

Ponyo Plants is a plant health app that identifies and analyzes a photo of your choice of plant through a machine learning model. Using Plant.id's plant analysis model, the app will first identify with a certain probability the plant you have uploaded, alongside displaying its health percentage. If you upload something that is NOT a plant (is_plant < 0.5), it will flash a warning asking you to upload a photo of a valid plant.

üì¶ Dependencies

Ponyo Plants uses Plant.id's API, which requires unique keys for access. If you do not have a personal Plant.id API key, create an account via this link.
Once registered, click on 'Create new API key'. A pop-up window will ask for the specifications of creating the API key; make sure to select 'Plant.id', name the API key, and assign the max number of credits (100). Copy that API key and place it inside your .env file as API_KEY=(your API key).
Ponyo Plants relies on Docker for containerization. If Docker isn't installed, download and install it from here.
